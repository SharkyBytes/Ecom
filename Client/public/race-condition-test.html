Flash Sale Race Condition Test

This page demonstrates race condition handling in flash sales. Open this page in multiple tabs to test simultaneous purchases.

Instructions

  1. Open this page in 3 separate browser tabs
  2. In the first tab, click "Create Flash Sale" to simulate a cancelled order
  3. In the other tabs, you should see the flash sale appear
  4. Click "Purchase" in multiple tabs simultaneously to test race condition handling
  5. Only one purchase should succeed, others should show appropriate error messages
